# Element UI Demo 项目

这是一个使用 Vite + Vue3 + Element Plus 构建的前端演示项目。

## 技术栈

- **Vite** - 快速的前端构建工具
- **Vue 3** - 渐进式 JavaScript 框架
- **Element Plus** - Vue 3 的桌面端组件库
- **TypeScript** - JavaScript 的超集，提供类型支持
- **Vue Router** - Vue.js 官方路由管理器
- **Pinia** - Vue 的状态管理库

## 项目特性

- ✅ Vite 构建工具，快速热重载
- ✅ Vue 3 Composition API
- ✅ TypeScript 支持
- ✅ Element Plus UI 组件库
- ✅ Vue Router 路由管理
- ✅ Pinia 状态管理
- ✅ ESLint + Prettier 代码规范

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 项目结构

```
src/
├── components/     # 组件目录
├── views/         # 页面组件
├── router/        # 路由配置
├── stores/        # Pinia 状态管理
├── assets/        # 静态资源
└── main.ts        # 应用入口文件
```
